# Тест типографики для казахского языка

## Проверка специальных символов казахского алфавита

### Заголовок с казахскими буквами
Сіздің оқу аймағыңыз

### Все специальные символы
ә і ң ғ ү ұ қ ө һ
Ә І Ң Ғ Ү Ұ Қ Ө Һ

### Примеры текста из интерфейса

#### Dashboard
- Сәлем (Привет)
- Сенің оқу аймағың (Твоя учебная зона)
- Күнделікті тапсырмалар (Ежедневные задания)
- Жылдам бастау (Быстрый запуск)

#### Courses
- Курстар картасы (Карта курсов)
- Оқу маусымы (Сезон обучения)
- Тарау (Глава)

#### Названия глав (из COURSE_TRANSLATIONS)
1. Тарау 1: Python-дағы алғашқы қадамдар
2. Тарау 2: Айнымалылар және түрлер
3. Тарау 3: if / else шарттары
4. Тарау 4: for / while циклдері
5. Тарау 5: Функциялар және параметрлер

#### Длинные слова для проверки переноса
- Бағдарламалау (Программирование)
- Айнымалылар (Переменные)
- Қайталау (Повтор)
- Салыстыру (Сравнение)

### Проверка в разных контекстах

#### В заголовках (h1)
# Сіздің оқу аймағыңыз

#### В подзаголовках (h2)
## Күнделікті тапсырмалар

#### В обычном тексте
Блок if выполняется только тогда, когда условие истинно.
if блогы шарт ақиқат болғанда ғана орындалады.

#### В кнопках и метках
- Бастау (Старт)
- Артқа (Назад)
- Сақтау (Сохранить)
- Толығырақ (Подробнее)

## Ожидаемый результат

✅ Все казахские символы (ә, і, ң, ғ, ү, ұ, қ, ө, һ) должны отображаться тем же шрифтом Noto Sans
✅ Нет визуальных различий в размере или толщине между казахскими и русскими буквами
✅ Длинные слова корректно переносятся на мобильных устройствах
✅ Шрифт одинаковый в заголовках (font-display) и обычном тексте

## Технические детали исправления

### 1. Замена шрифтов в index.html
- Было: Outfit (display), Nunito (sans)
- Стало: Noto Sans для обоих с поддержкой cyrillic-ext

### 2. Обновление styles.css
- Добавлена поддержка переноса слов для заголовков
- Добавлен класс .font-display с явным указанием Noto Sans

### 3. Локализация названий глав
- Курсы уже приходят локализованными с бэкенда через COURSE_TRANSLATIONS
- Удалена дублирующая логика локализации на фронтенде

### 4. Контент уроков
- Уже поддерживается через константу IS_KZ в Editor.tsx
- Динамическая генерация контента на основе языка
